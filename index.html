      Olá grande amigo e visitante, estou aqui para demonstrar um pouco do que tenho aprendido nesse curso de Domine Web - Jamilton Damasceno, este site por exemplo é o primeiro projeto do curso dele na parte de HTML. Tenho aprendido bastante sobre o conceito de tabelas e alguns pontos também sobre formulários. Coolspan serve para aglutinar colunas e o Rowspan serve para aglutinar linhas.

  • TR - Pode ser entendido como Table Row e cria uma linha de tabelas.

  • TD - Pode ser entendido como Table Data e seria o conteúdo de uma tabela.

  • TH - Pode ser entendido como Table Header e seria mais como um cabeçalho de tabela.
